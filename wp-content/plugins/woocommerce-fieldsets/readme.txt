This plugin adds the possibility to receive additional order info after the purchase has been made.
== Description ==
***Let customers add additional informtion to the order after payment.***

The _WooCommerce Forms & Fields plugin_ allows customers to add additional information to an order on the order detail page. This gives you the possibility to receive extra info from your customer, for example for extra shipping info, product customisation, etc.

The plugin adds a new side panel on the order detail page to allow shop managers to read the additional info. 

For more information, check out [our website](http://wordpress.geev.nl/product/woocommerce-fieldsets/).

This plugin is compatible with Wordpress 4.0 and WooCommerce 2.2.0-bleeding.

== Installation ==

1. Install WooCommerce Forms & Fields either via the Wordpress.org plugin directory or by uploading the files to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Upgrade Notice ==
Please backup first.
== Usage ==

Go to WooCommerce > Fieldsets, configure and start printing. Simple!
